시작하기 전에

이 가이드는 Windows 컴퓨터에 Claude Code를 설치하는 과정을 안내합니다. 프로그래밍 경험이 전혀 없어도 괜찮습니다. 화면에 보이는 대로 따라하면 됩니다.

설치할 것 3가지
Git
→
VS Code
→
Claude Code
순서대로 설치합니다. 약 20~30분 소요.

필요한 것:

— Windows 10 이상
— 인터넷 연결
— Claude 계정 (Pro, Max, Team, 또는 Enterprise)

각 프로그램이 왜 필요한지 간단히 설명하면:

Git — Claude Code가 내부적으로 사용하는 필수 부품입니다. 직접 쓸 일은 없지만 없으면 안 됩니다.
VS Code — 파일과 폴더를 보면서 작업할 수 있는 편집기입니다. 필수는 아니지만 강력 권장합니다.
Claude Code — 오늘의 주인공. AI가 여러분의 컴퓨터에서 직접 코드를 읽고 쓰고 실행합니다.

1

Git 설치

① 다운로드 페이지로 이동합니다.

아래 링크를 클릭하세요.

https://git-scm.com/downloads/win

페이지가 열리면 상단에 "Click here to download" 링크가 보입니다. 클릭하면 다운로드가 시작됩니다.

② 다운로드된 설치 파일을 실행합니다.

설치 과정에서 여러 화면이 나옵니다. 전부 기본값(Next)으로 진행하면 됩니다. 바꿀 것 없이 계속 Next → Next → Next → Install 입니다.

혹시 설치 중에 "Adjusting your PATH environment" 화면이 보이면, "Git from the command line and also from 3rd-party software"가 선택되어 있는지만 확인하세요. 기본값이 이거라 보통 이미 선택되어 있습니다.

③ 설치가 끝나면 확인합니다.

키보드에서 Windows 키를 누르고 "powershell"을 입력한 뒤 Windows PowerShell을 클릭해서 엽니다.

열린 창에 아래를 입력하고 Enter를 누르세요:

git --version

git version 2.xx.x 같은 숫자가 나오면 성공입니다.

"git을 찾을 수 없습니다" 같은 에러가 나오면 PowerShell을 닫았다가 다시 열어보세요. 그래도 안 되면 Git 설치를 다시 시도합니다.

2

VS Code 설치 — 권장

VS Code(Visual Studio Code)는 파일과 폴더를 보면서 작업할 수 있는 편집기입니다. Claude Code가 파일을 수정하면 바로 눈으로 확인할 수 있어서 훨씬 편합니다.

① 다운로드 페이지로 이동합니다.

https://code.visualstudio.com

큰 "Download for Windows" 버튼을 클릭하세요.

② 다운로드된 설치 파일을 실행합니다.

역시 기본값(다음)으로 쭉 진행합니다.

설치 중 "추가 작업 선택" 화면이 나오면, 아래 두 항목에 체크하는 걸 권장합니다:

✅ PATH에 추가 (기본 체크됨)
✅ "Code(으)로 열기" 작업을 파일 컨텍스트 메뉴에 추가

③ VS Code를 실행합니다.

설치가 끝나면 자동으로 VS Code가 열립니다. 혹시 한국어가 아니라면, 좌측 사이드바의 확장(Extensions) 아이콘을 클릭하고 "Korean Language Pack"을 검색해서 설치하세요.

3

PowerShell이란?

Claude Code를 설치하려면 PowerShell이라는 곳에 명령어를 입력해야 합니다. PowerShell은 컴퓨터에게 글자로 명령을 내리는 창입니다. 마우스로 클릭하는 대신, 키보드로 "이거 해"라고 타이핑하는 방식이죠.

PowerShell을 여는 방법은 두 가지가 있는데, 어디서 열든 똑같은 것입니다.

PowerShell 여는 법 — 두 가지 방법, 결과는 동일
방법 A
단독으로 열기
Windows 키 → "powershell" 검색 → Windows PowerShell 클릭
방법 B
VS Code 안에서 열기
VS Code 상단 메뉴 → 터미널 → 새 터미널
(또는 키보드 단축키: Ctrl + J)

핵심: 이 두 가지는 완전히 동일합니다.

VS Code 안에서 열리는 터미널도 PowerShell이고, 시작 메뉴에서 직접 여는 것도 PowerShell입니다. 어디서 열든 같은 명령어를 입력하면 같은 결과가 나옵니다.

다만 VS Code 안에서 열면 파일 목록을 옆에서 바로 볼 수 있어서 더 편합니다. 이 가이드에서는 방법 B(VS Code 안에서 열기)를 권장합니다.

4

Claude Code 설치

① PowerShell을 엽니다.

위에서 설명한 방법 A 또는 방법 B 중 편한 걸로 여세요.

② 아래 명령어를 복사해서 붙여넣고 Enter를 누릅니다.

irm https://claude.ai/install.ps1 | iex

복사: 명령어를 드래그한 뒤 Ctrl + C
붙여넣기: PowerShell 창에서 마우스 우클릭 또는 Ctrl + V

설치가 진행되면서 텍스트가 쭉 나옵니다. 끝날 때까지 기다리세요.

"이게 무슨 명령어인가요?"

irm은 인터넷에서 파일을 가져오는 명령어이고, iex는 가져온 파일을 실행하는 명령어입니다. 즉, Claude 공식 사이트에서 설치 스크립트를 다운로드해서 바로 실행하는 것입니다.

③ 설치가 끝나면 PowerShell을 닫았다가 다시 엽니다.

이 단계가 중요합니다. 새로 설치한 프로그램을 인식하려면 PowerShell을 반드시 새로 열어야 합니다.

PowerShell을 다시 열어도 claude 명령어가 인식되지 않으면, 컴퓨터를 재부팅한 뒤 다시 시도하세요. (특히 회사 PC에서는 재부팅이 필요한 경우가 많습니다.)

④ 설치를 확인합니다.

새로 연 PowerShell에 아래를 입력하고 Enter:

claude --version

버전 번호가 나오면 설치 성공입니다.

5

로그인

① PowerShell에서 아래를 입력하고 Enter:

claude

② 화면에 안내가 나타납니다. 지시를 따라가세요.

처음 실행하면 PowerShell 화면에 로그인 관련 안내가 표시됩니다. 화면의 지시를 따라가면 웹 브라우저가 열리고, Claude 로그인 페이지가 나타납니다. 본인의 계정으로 로그인하세요.

브라우저가 자동으로 안 열리면, PowerShell에서 C 키를 누르세요. 로그인 주소가 클립보드에 복사됩니다. 브라우저를 직접 열고 주소를 붙여넣으면 됩니다.

③ 브라우저에서 로그인이 완료되면, PowerShell로 돌아옵니다.

자동으로 로그인이 인식되고, Claude Code 입력 대기 상태가 됩니다. 로그인은 최초 1회만 하면 되고, 이후에는 자동으로 인증됩니다.

어떤 계정이 필요한가요?

Claude Code를 사용하려면 아래 중 하나가 필요합니다:

— Claude Pro 또는 Max (개인 유료 구독)
— Claude Team 또는 Enterprise (회사에서 제공하는 계정)

무료 계정으로는 사용할 수 없습니다.

6

설치 완료 확인

모든 설치가 끝났습니다. 마지막으로 전체가 잘 동작하는지 확인합니다.

① PowerShell에서 아래를 입력하고 Enter:

claude doctor

Claude Code가 자동으로 환경을 점검합니다. 문제가 있으면 무엇이 잘못되었는지 알려줍니다.

② VS Code에서 프로젝트 폴더를 열어봅니다.

VS Code를 열고 → 파일 → 폴더 열기 → 작업할 폴더를 선택합니다.
그다음 터미널을 열고 (Ctrl + J) claude를 입력하면, 해당 폴더를 기준으로 Claude Code가 시작됩니다.

축하합니다! 환경 설정이 완료되었습니다.

이제 Claude Code에게 무엇이든 시켜볼 수 있습니다.
